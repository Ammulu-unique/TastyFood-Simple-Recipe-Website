# TastyFoods

TastyFoods is a simple and clean recipe website created using **HTML and CSS**.  
The project displays traditional Indian sweet recipes with their ingredients and preparation steps in a visually organized layout.

---

## Features

- Responsive navigation bar
- Recipe sections with ingredients and process
- Image cards with hover effects
- Clean layout using Flexbox
- Simple and readable UI

---

## Tech Stack

- HTML5
- CSS3

---

## What I Learned

- Structuring webpages using semantic HTML
- Designing layouts with Flexbox
- Using CSS box model and shadows
- Adding hover effects and transitions
- Organizing content into reusable sections
